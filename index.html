Journée de la femme


Instructions :
 

Copier les deux fichiers white-ribbon.css et white-ribbon.svg dans le
     'root' (racine) de votre espace internet


Copier le code dans la page web où vous souhaitez voir le ruban apparaître.


dans la section <head> : 


     <link rel="stylesheet" type="text/css" href="white-ribbon.css">


dans la section <body>:


     <div id="white-ribbon"><a href="http://8mars.info/">
     <img src="white-ribbon.svg"  alt="ruban pour la journée 
     internationale de la femme"></a></div>
